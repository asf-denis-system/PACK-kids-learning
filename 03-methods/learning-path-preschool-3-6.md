# Learning Path: Дошкольный трек (3–6 лет)

**Назначение:** Стартовая траектория для взрослого-наставника по нулевым принципам с дошкольником.
**Читай это, если:** Выбран трек 3–6 и нужен управляемый маршрут на 2-3 первых цикла.

**Навигационная цепочка:** [Manifest](../00-pack-manifest.md) → [Ontology](../ontology.md) → [Карточки эпизодов](preschool/) → этот файл

---

## Цель трека

Дошкольник начинает:
1. видеть части и целое, разбирать и пересобирать;
2. различать, что осталось тем же при внешних изменениях;
3. выбирать «лучше» по правилу, а не по произвольному вкусу.

Целевой критерий на каждый цикл: **спонтанный перенос** в новой ситуации без подсказки взрослого.

---

**Перед первым циклом (20–30 мин):**

1. Прочитать [00-pack-manifest.md](../00-pack-manifest.md) — понять что это за программа.

## Weekly Loop (для каждого цикла)

### Шаг 0. Настройка цикла (один раз в начале)

1. Выбрать один Z-принцип из последовательности выше.
2. Подготовить 2–3 бытовых контекста для эпизодов.
3. Открыть карточку эпизода из `preschool/`.
4. Зафиксировать в Family Exocortex: текущий принцип, дата старта.

### Шаг 1. Планирование недели

1. Поставить 3–4 коротких слота (5–15 мин каждый).
2. Привязать к уже существующим бытовым ситуациям — не создавать отдельное «время».
3. Один фокус на неделю — не вводить второй принцип параллельно.

### Шаг 2. Эпизоды

1. Проводить эпизод строго по карточке (7 шагов).
2. После каждого — одна короткая запись в [04-work-products/01-weekly-episode-log-template.md](../04-work-products/01-weekly-episode-log-template.md).
3. Если форма ломается (ребёнок сопротивляется, взрослый давит) → упростить один параметр, не бросать цикл.

### Шаг 3. Обзор недели

1. Заполнить Weekly Episode Log.
2. Зафиксировать одно наблюдение про ребёнка.
3. Зафиксировать одно наблюдение про взрослого.
4. Принять одно решение на следующую неделю.

### Шаг 4. Закрытие цикла

1. Заполнить [04-work-products/02-mastery-map-template.md](../04-work-products/02-mastery-map-template.md).
2. Заполнить [04-work-products/04-adult-reflection-template.md](../04-work-products/04-adult-reflection-template.md).
3. Принять одно из решений:
   - продолжаем тот же принцип (если переноса ещё нет);
   - расширяем контексты (если перенос в одном месте, но не в других);
   - переходим к следующему принципу (см. критерии ниже).

---

## Сигналы готовности к переходу

Переходим к следующему принципу, если к концу 2–4 недель:

1. Ребёнок включается без лишнего сопротивления.
2. Есть 2–3 наблюдения самостоятельного применения в **разных** бытовых контекстах.
3. Взрослый может честно назвать, что именно ребёнок начал различать.

Остаёмся на том же принципе, если:
- перенос есть только в одном контексте;
- ребёнок действует в основном с подсказкой;
- у взрослого растёт давление или усталость;
- weekly log показывает срыв формы.

---

## Антипаттерны

1. Открывать 2–3 принципа одновременно «для ускорения».
2. Переходить к следующему принципу по ощущению «вроде уже понял», без фактов переноса.
3. Гнаться за количеством эпизодов вместо устойчивости ритма.
4. Создавать отдельное «учебное время» вместо встраивания в бытовые ситуации.
