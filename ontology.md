## Ключевые термины (Ubiquitous Language)

| Термин | Определение |
|--------|-------------|
| **Эпизод** | Конкретное выполнение практики: тема → вопрос → версии → проверка (5–15 мин) |
| **Перенос (трансфер)** | Применение способа рассуждения в новом контексте без подсказки |
| **Нулевые принципы** | Базовые различения Z0–Z7 без которых невозможно рассуждение |
| **FP-ядро** | Правила валидного рассуждения: наблюдение, проверка версий, согласование критериев |
| **Цикл** | Единица работы по одному принципу (обычно 3–6 недель) |
| **Guard Policy** | Критерии перехода к следующему принципу (уровень 3 ≥ 3 раза, перенос в 2+ контекстах) |
| **Mastery Map** | Карта прогресса по Z0–Z7 (уровни 0–3 по каждому принципу) |
| **Ресурс ребёнка** | Наблюдаемые условия: внимание, эмоции, устойчивость — не диагноз |

## Иерархия принципов (место Pack в системе)

```
ZP (Нулевые принципы)
    ↑
FPF (Первые принципы + фреймворк)
    ↑
SPF (Форма для вторых принципов)
    ↑
KIDS-PACK (Вторые принципы для детей) ← МЫ ЗДЕСЬ
    ↑
Конкретная семья (3-й принципы / применение)
```
